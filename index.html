Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • H1, H2, H3, H4, H5, and H6 are all used for headers, 1 being the largest and only used once.
  • ul is an unordered list and li creates the bullet points
  • Everything gets closed out with a forward slash.

CSS

  • CSS is used to add visual elements to the website.

Git

  • Git and Git Hub are used to work on webdevelopment.
  • Git is local to the computer and Git Hub is on the cloud.
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

JavaScript

  • Java is used to add interactive elements to a site.
